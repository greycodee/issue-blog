## 安装 zsh 

首先执行下面命令

```bash
$ zsh --version                                                                            
zsh 5.9 (x86_64-pc-linux-gnu)
```
如果输出了 zsh 的版本号，则说明已经安装了 zsh 可以跳过此步骤

Manjaro 安装命令

```bash
sudo pacman -S zsh
```

更多安装命令请查看：https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH#how-to-install-zsh-on-many-platforms

## 设置默认 Shell 为 zsh

查看当前SHELL

```bash
$ echo $SHELL                                                                          
/bin/bash
```
Manjaro 设置命令：

```bash
$ chsh -s $(which zsh)
```
