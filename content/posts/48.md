在tmux外的 `TERM` 的变量值是：

```bash
❯ echo $TERM
xterm-256color
```

在tmux内的值是：

```bash
❯ echo $TERM
screen
```

## 问题

导致在tmux中显示的颜色和真正终端的颜色有区别

## 解决方法

在 `~/.tmux.conf` 文件中配置如下配置（没有这个文件就创建一个）

```bash
set -g default-terminal "xterm-256color"
```

然后关闭 tmux 重新打开，就恢复正常了
