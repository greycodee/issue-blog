### neovim 快捷键

```bash
# 窗口切换-切换到左边
ctrl+w  h

# 切换到右边
ctrl+w  l
```


### tmux 快捷键

```bash
# 左右-分割屏幕
ctrl+b  %

# 上下-分割屏幕
ctrl+b  "

```

### 命令

```bash
:NvimTreeToggle    #在 neovim 界面打开 nvim-tree
```
