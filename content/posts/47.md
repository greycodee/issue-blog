### neovim 快捷键

```bash
# 窗口切换-切换到左边
ctrl+w  h

# 切换到右边
ctrl+w  l
```


### tmux 快捷键

```bash
# 左右-分割屏幕
ctrl+b  %

# 上下-分割屏幕
ctrl+b  "

# 窗口切换
ctrl+b  方向键⬆️⬇️⬅️➡️ 进行窗口上下左右的切换

# 关闭当前窗口
ctrl+b  x

# 查看当前所有窗口
ctrl+b  w

```

### 命令

```bash
:NvimTreeToggle    #在 neovim 界面打开 nvim-tree
```
